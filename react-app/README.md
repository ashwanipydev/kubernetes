# k8s45 React App (TypeScript)

This is a minimal CRA-like React + TypeScript starter included for the Kubernetes 45-day repo.

Features
- Responsive grid UI
- Dark / Light theme toggle persisted to `localStorage`
- Simple day-wise progress tracker (45 days)

Run locally
1. cd react-app
2. npm install
3. npm start

Build & deploy to GitHub Pages
- `npm run build` then `npm run deploy` (requires `gh-pages` and branch setup)
