Docker Playground (Simulator)

Practice common Docker CLI commands in a lightweight simulator, or open an actual session on Play with Docker.

Open Play with Docker

Tasks

  1. Pull the nginx image: docker pull nginx
  2. Run nginx named web1: docker run --name web1 nginx
  3. Stop web1: docker stop web1
  4. Remove web1: docker rm web1

Tip: run help for a list of supported commands in this simulator.
