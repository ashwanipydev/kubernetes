# Kubernetes — 45 Day Learning Path

This repository contains a 45-day Kubernetes learning path (Option B): an `index.md` and per-day directories under `days/` with hands-on labs, weekly projects under `projects/`, and a static progress tracker under `progress-tracker/`.

Branch for scaffolding: `scaffold/45dayk8s` (created by automation)

See `index.md` for the Table of Contents and `progress-tracker/` for the GitHub Pages–deployable tracker.
