# Images Directory

This directory contains images used throughout the 45 Days of Kubernetes course.

## Required Images

Place the following images in this directory:

- `kubernetes-logo.png` - Main Kubernetes logo for README
- `architecture-diagram.png` - Kubernetes architecture diagram
- `pod-diagram.png` - Pod structure diagram

## Image Guidelines

- Use PNG format for diagrams
- Keep file sizes under 500KB
- Use descriptive filenames
